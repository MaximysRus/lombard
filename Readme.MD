# Модуль "Залог / Выкуп"

Веб-приложение для сети ломбардов. Реализует функционал создания залогов (с динамическими характеристиками товаров в зависимости от тарифа) и выкупа залогов с расчетом пени за просрочку. 
Проект полностью контейнеризован и готов к локальному запуску одной командой.

## Технологический стек

* **Backend:** Node.js, NestJS, TypeScript
* **Database:** PostgreSQL + Prisma ORM
* **Frontend:** React (Vite), TypeScript, Material-UI (MUI v6), Zustand
* **Infrastructure:** Docker Compose (с healthcheck'ами баз данных)
* **Testing:** Jest, Supertest (E2E)

---

## Как запустить проект

1. Убедитесь, что у вас установлены **Docker** и **Docker Compose**.
2. Склонируйте этот репозиторий
3. В корне выполните команду `docker compose up --build -d`
4. Дождитесь запуска контейнеров и накатывания миграции ~15 секунд
5. Фронт проекта доступен по адрессу http://localhost:5173/


## Как запустить e2e тест

### Локально

1. Убедитесь, что у вас установлен **Node** не ниже v24.18.0 версии
2. Удостовертесь, что у вас запущены контейнеры 
3. Зайдите в backend (cd backend в корне)
4. Выполните команду `npm run test:e2e:full`
5. Команда выполнит миграции в тесовую базу, а затем сам тест

### В контейнере

1. Зайдите в контейнер backend
2. Запустите `npm run test:e2e:full`
